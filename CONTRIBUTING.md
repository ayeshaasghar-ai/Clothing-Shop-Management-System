# 🤝 Contributing to Clothing Shop Management System

Thank you for your interest in contributing! We welcome all contributions — bug fixes, new features, improvements, and suggestions.

---

## 📋 Before You Start

- Make sure you have **JDK 25** and **Maven** installed for backend
- Make sure you have **VS Code** or any code editor installed
- Have a **GitHub account**

---

## 🚀 How to Contribute

### 1️⃣ Fork the Repository
- Go to the repository on GitHub
- Click the **"Fork"** button (top right)
- This creates a copy of the project on your GitHub account

### 2️⃣ Clone Your Fork

    git clone https://github.com/your-username/Clothing-Shop-Management-System.git

### 3️⃣ Create a New Branch

    git checkout -b feature/your-feature-name

Example: feature/add-search-filter or fix/login-bug

### 4️⃣ Make Your Changes
- Make your changes in the code
- Test everything works properly before submitting

### 5️⃣ Commit Your Changes

    git add .
    git commit -m "Brief description of what you changed"

### 6️⃣ Push to GitHub

    git push origin feature/your-feature-name

### 7️⃣ Open a Pull Request
- Go to your forked repository on GitHub
- Click **"Compare & pull request"**
- Write a clear description of what you changed and why
- Click **"Create pull request"**

---

## 🐛 Reporting Bugs

Found a bug? Please open an **Issue** on GitHub with:

- A clear title describing the problem
- Steps to reproduce the bug
- What you expected to happen
- What actually happened
- Screenshots (if possible)

---

## 💡 Suggesting Features

Have an idea? Open an **Issue** and label it as `enhancement`. Describe:

- What feature you want
- Why it would be useful
- How you think it should work

---

## ✅ Code Guidelines

- Keep your code clean and readable
- Add comments where necessary
- Test your changes before submitting
- One feature or fix per pull request

---

## 📬 Contact

If you have any questions, feel free to open an issue and we'll get back to you!
